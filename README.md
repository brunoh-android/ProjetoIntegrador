# ProjetoIntegrador 

## Projeto em Kotlin - lista filmes por categorias

## Bibliotecas Utilizadas:
    
    - Retrofit
    - OkHttp
    - Coroutines
    - Fragment KTX
    - Livedata KTX
    - Glide
    - Firebase BOM
    - Room Database

    
## API - "https://api.themoviedb.org"
